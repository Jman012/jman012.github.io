This is the API

getCreator

Retrieve detailed information about a specific creator.

URL

https://www.floatplane.com/api/v3/creator/info

HTTP Method

GET

Response Type

CreatorModelV3

Parameters

  • Query: id String

    The GUID of the creator being searched.

getCreators

Retrieve and search for all creators on Floatplane. Useful for creator discovery and filtering.

URL

https://www.floatplane.com/api/v3/creator/list

HTTP Method

GET

Response Type

List

Parameters

  • Query: search String

    Optional search string for finding particular creators on the platform.
