This is the API

getCreatorInfoByName

Retrieve detailed information on one or more creators on Floatplane.

URL

https://www.floatplane.com/api/v2/creator/named

HTTP Method

GET

Response Type

List

Parameters

  • Query: creatorURL List(String)

    The string identifer(s) of the creator(s) to be retrieved.

getInfo

Retrieve detailed information on one or more creators on Floatplane.

URL

https://www.floatplane.com/api/v2/creator/info

HTTP Method

GET

Response Type

List

Parameters

  • Query: creatorGUID List(String)

    The GUID identifer(s) of the creator(s) to be retrieved.
