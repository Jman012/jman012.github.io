This is the API

getActivityFeedV3

Retrieve recent activity for a user, such as comments and other interactions they have made on posts for creators.

URL

https://www.floatplane.com/api/v3/user/activity

HTTP Method

GET

Response Type

UserActivityV3Response

Parameters

  • Query: id String

    The GUID of the user being queried.

getExternalLinksV3

Retrieve configured social media links from a user's profile.

URL

https://www.floatplane.com/api/v3/user/links

HTTP Method

GET

Response Type

Map

Parameters

  • Query: id String

    The GUID of the user being searched.

https://www.floatplane.com/api/v2/user/ban/status

Response Type
